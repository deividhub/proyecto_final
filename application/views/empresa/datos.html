¿Para que queremos tus datos? Todos los datos que nos estan proporcionando como usuario se almacenan en una base de datos(Exceptuando los datos bancarios que no se almacenan) con el fin de poder realizar envios a la dirección que se nos proporciona. Estos datos en ningún caso van a ser utilizados por terceros, exceptuando la empresa de mensajería que utilizará estos datos para llegar a tu domicilio.
